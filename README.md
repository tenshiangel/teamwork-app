## Setup

Make sure to install the dependencies

```bash
composer install
npm install

(note: currently using npm v14.21.3)
```

## Development

Start the development server on http://localhost:6800

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```
